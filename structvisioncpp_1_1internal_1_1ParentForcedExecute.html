Detailed Description

to check whether or not the parent of the Expr requires that the expr to be executed and the LeafNode as a result to be returned. True means that execute the expression and return the LeafNode; false means return the execute as it is. This is an extra check making sure that we wont break the tree when it is not necessary. Such example would be when you have reduction as a root of a subexpression tree in the middle of an expression, it is possible to minimise the number of subexpression to be creating by 2.
