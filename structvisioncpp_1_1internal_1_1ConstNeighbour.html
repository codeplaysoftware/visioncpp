Detailed Description

template<typename T>
struct visioncpp::internal::ConstNeighbour< T >

ConstNeighbour is used to provide global access to the constant memory. It is used as an input type for user functor when a constant pointer needed to be passed on the device side. An example of such node can be a filter node for convolution operation. template parameters.
