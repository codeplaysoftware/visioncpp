struct visioncpp::internal::SubExprExecute< Val, ExecPolicy, LC, LR, LCT, LRT, Expr, DeviceT >

it is used to statically determine whether or not a subexpression execution is needed. It increases the execution time by avoiding executing the subexpression when it is not needed. Using this struct with sub_expression_evaluation parameter in every non-terminal node, it is possible to determine such condition at compile time.
