Detailed Description

template<size_t Memory_Type, size_t N, size_t C, size_t R, size_t LVL, size_t Sc>
struct visioncpp::internal::LeafNode< PlaceHolder< Memory_Type, N, C, R, Sc >, LVL >

The specialisation of LeafNode for the PlaceHolder.

It is used to store the information of visionMemory. template parameters:

Parameters
Memory_Type: determines the type of the memory {Buffer2D, Buffer1D, Host, Image}
Nis the position of the leafNode in the expression tree
Cis the column size of the visionMemory
Ris the row size of the visionMemory
LVLrepresents the level of the node in the expression tree.
