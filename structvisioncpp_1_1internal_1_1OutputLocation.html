Detailed Description

template<bool IsRoot, size_t OutOffset>
struct visioncpp::internal::OutputLocation< IsRoot, OutOffset >

This is used to find whether a node should use a global memory output or a local memory output is created for that node. When the node is the immediate child of the root (Assign or Partial Assign) we do not create the local shared memory for it and directly save the result in the global memory. In this case we can avoid creation of unnecessary shared memory. template parameters.

Template Parameters
IsRootboolean value represent if the expression is the immediate child of a root node
OutOffsetthe offset for location of the local memory if it exists
