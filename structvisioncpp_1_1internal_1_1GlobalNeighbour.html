Detailed Description

template<typename T>
struct visioncpp::internal::GlobalNeighbour< T >

GlobalNeighbour is used to provide local access for each element of the global memory based on the Coordinate passed by eval expression. It is used as an input type for user functor when the global neighbour operation is required. template parameters.
