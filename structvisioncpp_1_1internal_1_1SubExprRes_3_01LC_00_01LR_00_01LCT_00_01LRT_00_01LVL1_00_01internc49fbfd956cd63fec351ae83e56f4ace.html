Detailed Description

template<size_t LC, size_t LR, size_t LCT, size_t LRT, typename RHS, size_t LVL1, size_t LVL2, typename DeviceT>
struct visioncpp::internal::SubExprRes< LC, LR, LCT, LRT, LVL1, internal::LeafNode< RHS, LVL2 >, DeviceT >

specialisation of the SubExprRes when it is a LeafNode.

It does nothing but returns the given input leaf node as there is no operation to apply. This specialisation is used to speed up the process and to prevent extra memory creation and execution.
