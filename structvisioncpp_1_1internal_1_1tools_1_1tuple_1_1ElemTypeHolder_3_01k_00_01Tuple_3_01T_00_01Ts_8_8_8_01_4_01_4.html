Detailed Description

template<size_t k, class T, class... Ts>
struct visioncpp::internal::tools::tuple::ElemTypeHolder< k, Tuple< T, Ts... > >

Specialisation of the ElemTypeHolder class when the number of the elements inside the tuple is bigger than 1.

It recursively calls itself to detect the type of each element in the tuple.

Template Parameters
T: the type of the first element in the tuple.
Ts...the rest of the elements in the tuple. Ts... can be empty.
Kis the Kth element in the tuple.
