struct visioncpp::internal::Fill< LeafNode< PlaceHolder< memory_type::Const, N, Cols, Rows, Sc >, LVL >, Loc, Params... >

Partial specialisation of the Fill when the LeafNode contains the const variable.

In this case we load nothing in to the shared memory as there is no shared memory for const variable and the const variable directly copied to the device and accessed by each thread.
