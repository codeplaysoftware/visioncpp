Detailed Description

template<size_t IsRoot, size_t LC, size_t LR, size_t LVL, typename RHS>
struct visioncpp::internal::LocalOutput< false, IsRoot, LC, LR, LeafNode< RHS, LVL > >

LocalOutput specialisation for leaf node it creates the local accessor that contains the data of the leaf node from global memory.

This local memory then will be used by other non-terminal node as an input value.
