Detailed Description

This file contains the VirtualMemory struct. VirtualMemory struct is nothing but a future LeafNode representing the result of the subexpression passed to it to be executed. It is used by user to schedule any arbitrary subexpression to be executed in a separate kernel. This will allow a user to manually break the expression tree at compile time.
