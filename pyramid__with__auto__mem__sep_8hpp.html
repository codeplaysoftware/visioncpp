Detailed Description

This file contains the construction of the pyramid node where a user can pass separable filter2d functor and general downsampling functor to execute. Here, the user does need to pass the output memory. The output memory will be created based on the depth of pyramid. Also an output file will be created which merge all the result of the pyramid image in one file.
